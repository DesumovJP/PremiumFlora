# Premium Flora CMS

Преміальна CMS для оптового магазину квітів: POS-термінал, каталог товарів, клієнти та аналітика. Мобайл-first, шрифти Geist, компоненти у стилі shadcn/ui.

## Запуск

```bash
cd frontend
yarn install
yarn dev
```

Потім відкрийте [http://localhost:3000](http://localhost:3000).

## Стек та дизайн
- Next.js 16 (app router), React 19, TypeScript, Tailwind CSS v4.
- Компоненти у стилі shadcn (button, badge, card, select, tabs, table, sheet, scroll-area) з Radix UI та class-variance-authority.
- Тема: преміальний зелений, радіальні градієнти, мікрообводки, заокруглення 12–20px, мобайл-first сітки.

## Структура
- `app/page.tsx` — головна сторінка з табами: POS, Товари, Клієнти, Аналітика.
- `components/ui/*` — бібліотека атомів/молекул (button, select, card, badge тощо).
- `lib/utils.ts` — утиліта `cn` (clsx + tailwind-merge).
- `app/globals.css` — палітра, фон, базова типографія.

## Демо-дані
- Товари з варіантами (розмір/ціна/залишок), попередньо заповнений кошик та клієнти.
- Аналітичні KPI, топ-товари, планування поставок, історія замовлень.

## UX нотатки
- Мобайл-first: сітки стають стовпчиком, навігація через Tabs, на десктопі — сайдбар.
- Видимість стану: бейджі для запасів/статусів, підсвітка primary дій, чітка ієрархія шрифту.
- Безпечні дії: вторинні кнопки в обводці, небезпечні — іконка `Trash`.
